# 免费在线工具集

一个免费、安全、隐私保护的在线工具合集，所有处理均在浏览器本地完成。

## 工具列表

| 工具 | 说明 | 状态 |
|------|------|------|
| 🖼️ [在线图片压缩](/image-tools/) | 免费压缩 PNG/JPG/WebP 等格式图片 | ✅ 已上线 |
| 📄 [在线 PDF 工具](/pdf-tools/) | PDF 合并、拆分、压缩 | ✅ 已上线 |
| ✏️ 在线文字处理工具 | 字数统计、文本转换、JSON格式化 | 🔄 开发中 |

## 技术栈

- 纯 HTML/CSS/JavaScript
- Tailwind CSS
- pdf-lib (PDF 处理)
- Canvas API (图片处理)
- 全部客户端处理，不上传服务器
