什么是身份证号码？

中国大陆第二代居民身份证由18位数字组成（最后一位可能为X），具有唯一性和终身不变性。身份证号码包含了公民的户籍所在地、出生日期、性别和校验码等信息。

身份证号码编码规则

  • 前1-2位：省级行政区代码（如 11=北京市, 31=上海市, 44=广东省）
  • 3-4位：地级行政区代码（城市）
  • 5-6位：县级行政区代码（区/县）
  • 7-14位：出生日期（YYYYMMDD）
  • 15-17位：顺序码（第17位奇数=男性，偶数=女性）
  • 第18位：校验码（0-9或X，使用ISO 7064:1983 MOD 11-2算法计算）

校验码计算方法

将前17位数字分别乘以对应的加权因子 [7, 9, 10, 5, 8, 4, 2, 1, 6, 3, 7, 9, 10, 5, 8, 4, 2]，求和后除以11取余数，根据余数查表得到校验码：余数0→1, 1→0, 2→X, 3→9, 4→8, 5→7, 6→6, 7→5, 8→4, 9→3, 10→2。
