Sobre o CEP

O CEP (Código de Endereçamento Postal) é um sistema de código postal brasileiro criado pelos Correios. Possui 8 dígitos no formato #####-###. Esta ferramenta contém uma base de dados com mais de 50 CEPs das principais cidades do Brasil.
